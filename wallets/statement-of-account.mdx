A statement of account is a formal summary of all activity on a wallet over a specified date range. Use it for accounting, reconciliation, regulatory reporting, or sharing with your finance team.

## Generate via dashboard

1. Go to the **Balance** tab in [app.fincra.com](https://app.fincra.com)
2. Select the currency wallet
3. Set your date range
4. Download the statement

## Generate via API

```bash
curl --request GET \
  --url 'https://api.fincra.com/wallets/logs/statement?currency=NGN&dateFrom=2026-06-01&dateTo=2026-06-30' \
  --header 'api-key: YOUR_API_KEY'
```

<Info>
  Statement endpoint: `GET /wallets/logs/statement`. `currency`, `dateFrom`, and `dateTo`
  are **required** query parameters; `perPage` max is 50. The response carries opening and
  closing balances plus the itemized entries, inside the standard success envelope.
</Info>

## What's in a statement

A wallet statement includes all credits and debits for the selected currency over the chosen date range:

- Opening and closing balance
- Itemized transaction log (payments in, payouts out, conversions, top-ups, wallet transfers)
- Transaction references and timestamps
- Net movement for the period
